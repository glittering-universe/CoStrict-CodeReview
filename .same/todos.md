# CoStrict Code Review 项目任务列表

## 项目概述

CoStrict 是一个基于 TypeScript + React + Bun 的智能代码审查 Web 应用，提供现代化的用户界面和强大的代码审查功能。

## 开发规范

遵循 [`.roo/rules-code/coding-standards.md`](.roo/rules-code/coding-standards.md) 中定义的编码规范：
- 使用 PascalCase 命名组件
- 保持组件单一职责
- 使用 TypeScript 严格模式
- 遵循状态管理最佳实践
- 使用相对路径进行导入

## 注意事项

- 使用 Bun 作为包管理器
- 开发前运行 `bun install`
- 提交前运行 `bun run lint` 和类型检查
- 遵循 Conventional Commits 规范
- 所有更改需通过测试验证
